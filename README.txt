> Rodar o server do mongoDB localmente:

1. Abrir um terminal
2. Navegar até a pasta do mongodb (C:\Program Files\MongoDB\Server\3.6\bin)
3. Executar o seguinte comando com a flag --dbpath e a pasta que eu escolhi para o BD:
mongod --dbpath "D:\Leo\Desktop\Node js\MongoDB\data\db"
